# 📚 자료구조 실습: Fork & Pull Request 연습

이 저장소는 자료구조 수업의 Git 협업 실습을 위한 공간입니다.

### 🛠 실습 미션
1. 이 저장소를 본인 계정으로 **Fork** 합니다.
2. 본인 계정의 저장소에서 `students/` 폴더 안에 파일을 생성합니다.
3. 파일명은 반드시 `학번_이름.txt` 로 합니다.
4. 파일 내용은 간단한 자기소개와 이번 학기에 도전해보고 싶은 실습 프로젝트 등 을 적습니다.
5. 완료 후 이 원본 저장소로 **Pull Request(PR)**를 보냅니다.

### ⚠️ 주의사항
- `students/` 폴더 외의 다른 파일은 수정하지 마세요.
- PR 제목은 `[실습] 학번 이름 제출합니다` 형식을 지켜주세요.
